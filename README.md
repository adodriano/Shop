# SHOP
API para cadastro de Produtos e Categorias

# Tecnologias e ferramentas utilizadas
- Visual Studio 2022
- ASP.NET Core 7
- EF Core
- Documentação SWAGGER
- SQL SERVER
- JWT
- Deploy AZURE

# Funcionalidades
- Cadastro, Listagem, Detalhes, Atualização, e Remoção de Categorias e Produtos
- Autenticação e Autorização
